# OckJuYong.github.io

 - MZ 단어사전 -
최근 많아진 MZ에 관련된 내용들 그중 MZ 세대가 사용하는 신조어등을 이해하기 쉽도록 단어사전형태로 만들어 놓은 프로그램입니다.

백엔드에서 MZ단어, 로그인 정보등을 받아오고 로그인 정보가 있다면 MZ는 현 세대이기때문에 지속적으로 변하는 신조어 추가를 위해 사용자도 단어 추가 신고등을 할 수 있도록 해두었습니다. 3명이상 해당 단어 신고시 삭제가 되도록 만들었습니다. 프론트엔드에서는 해당 필요에 따른 페이지 전환 디자인 백엔드에서 정보 가져와서 보여주기등등을 맡아서 진행하였습니다.
